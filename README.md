# Gunit Chawla - Extraordinary Portfolio

This is a 3D immersive portfolio built with Next.js, React Three Fiber, and Tailwind CSS.

## Features

- **3D Starfield Background**: A procedural particle system that creates a sense of depth and space.
- **Hero Section**: Animated text reveals and a scroll indicator.
- **About Section**: Glassmorphism cards with tilt effects.
- **Skills Section**: A 3D floating cloud of skills using `Text` from `@react-three/drei`.
- **Projects Section**: 3D cards showcasing work.
- **Contact Section**: A functional form with a 3D holographic Earth model.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **3D**: Three.js, React Three Fiber, Drei
- **Animations**: Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
